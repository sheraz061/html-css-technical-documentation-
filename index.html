HTML and CSS

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are core technologies for building websites.

HTML provides the structure of web pages, while CSS is used to control the layout and presentation.

  • HTML uses tags to structure content.
  • CSS specifies how the content is displayed.
  • HTML and CSS work together to create visually appealing web pages.
  • Learning HTML and CSS is fundamental for web development.
  • HTML elements include headings, paragraphs, links, images, etc.

HTML tags:

<html> <head> <title> <body> <div>
JavaScript and Java

JavaScript and Java are different programming languages.

JavaScript is a scripting language used to create interactive effects within web browsers.

  • Java is a programming language used to create applications and programs.
  • JavaScript is used to create dynamic and interactive web pages.
  • Java is used for building desktop, web, and mobile applications.
  • JavaScript code is run in the browser.
  • Java code is compiled and run on the server.

Python

Python is a high-level programming language used for general-purpose programming.

Python emphasizes code readability and its syntax allows programmers to express concepts in fewer lines of code compared to other languages.

  • Python is widely used for web development, scientific computing, AI, and more.
  • Python is known for its simplicity and ease of learning.
  • Python's standard library is extensive.
  • Python supports multiple programming paradigms.
  • Python has a large and active community.

Python examples:

print() def import for ... in if ... else
SQL

SQL (Structured Query Language) is a domain-specific language used for managing data in a relational database.

SQL syntax is similar to English, making it easy to learn.

  • SQL is used for database management and data manipulation.
  • Common SQL commands include SELECT, INSERT, UPDATE, DELETE.
  • SQL is essential for working with databases.
  • SQL supports transactions and data integrity.
  • SQL is used in many database systems, like MySQL, PostgreSQL, SQLite, etc.

SQL examples:

SELECT INSERT INTO UPDATE DELETE CREATE TABLE
Git and GitHub

Git is a distributed version control system for tracking changes in source code during software development.

GitHub is a web-based platform that provides hosting for version control using Git.

  • Git allows multiple developers to collaborate on projects.
  • GitHub provides additional features like issue tracking, pull requests, and project management tools.
  • Git and GitHub are essential for modern software development.
  • Git commands include init, clone, add, commit, push, pull.
  • GitHub is widely used in open-source projects.

Git examples:

git init git clone git add git commit git push
